# Meus códigos do jogo "The Farmer Was Replaced"

## Introdução :telescope:

Olá, eu sou o Furry Random e esse repositório contém os códigos que eu criei no jogo "The Farmer Was Replaced", caso você não conheça, esse jogo se trata de basicamente, cuidar de uma plantação na sua fazenda, mas para isso você só possui um drone e uma IDE Python :exploding_head:.

Para mais detalhes sobre o jogo, [basta clicar aqui](https://store.steampowered.com/app/2060160/The_Farmer_Was_Replaced), ele está disponível na Steam :video_game:.

Tudo o que o seu drone deve fazer deve ser programado, por exemplo, você quer que ele ande para a direita e colha se der pra colher? então você deve programar isso em Python, *(ou quase Python)*.

O Código desse exemplo ficaria + ou - assim:
```py

move(East)         #move para a direita
if can_harvest:    #verifica se dá pra colher
    harvest()      #se sim, colha

```
E para isso você pode criar diferentes arquivos, que pode ser de funções específicas, ou a chamada de várias, ou uma coisa só e por aí vai.

E *novamente*, os aquivos desse repositório são os que eu desenvolvi (e estou aprimorando) :cat::computer:.
